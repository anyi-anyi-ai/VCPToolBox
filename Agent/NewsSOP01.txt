# 《兽耳日报》标准化制作流程 (SOP) v3.1
——由 Sola.txt 核心协议与芙铃 v1.0 实践经验提炼——

## 一、Phase 0: 情报预加载
在生产日报前，必须先消化以下系统自动注入的数据源：
- `{{VCPDailyHot}}` — 多平台热榜聚合
- `{{ArxivDailyPapersData}}` — ArXiv 学术论文追踪
这些占位符是确定当日选题方向和优先级的**第一手线索**。

## 二、Phase 1: 情报搜集
使用 `TavilySearch` / `VSearch` 并行搜索，覆盖以下板块：

| 板块 | TavilySearch topic | 关键词方向 |
|---|---|---|
| 国际政治 | news | 地缘冲突、外交动态、制裁、选举 |
| 产业经济 | finance | 市场趋势、央行政策、产业链变动、中国经济 |
| 科学前沿 | research | Nature/Science 新成果、生命科学、物理、天文 |
| AI/科技动态 | general | AI、大模型、开源、芯片、智能体、科技公司 |
| 文化与游戏 | general | 按当日热点灵活选取 |

**每期必备的两个"非新闻"素材：**
1. **【每日艺术】（本地资产优先）**：
   - **首选方案**：调用 `FileOperator(ListDirectory)` 扫描本地图库 `H:\VCP\VCPzhangduan\VCPToolBox\image\DailyArt\`。
   - **随机抽卡**：从图库下的画家子目录中随机挑选一张画作。
   - **URL 映射**：将文件名拼接为图床 URL：`http://localhost:6005/pw=666888/images/DailyArt/[画家目录]/[文件名]`。
   - **撰写文案**：根据画作内容和风格撰写一段与当日日报氛围呼应的短评。
   - **备用方案**：若本地图库为空或访问失败，则回退为 `TavilySearch` 在线搜索。

2. **【每日诗文】**：选取一首短诗或一段文学摘录，作为日报的精神收尾。优先东西方经典，也可选当代佳作。附简要中文翻译。

**信息筛选原则：**
- **宁缺毋滥**：5条精选胜过15条堆砌。
- 每条新闻必须包含：**事实核心** + **品茗** (为什么值得关注)。
- 如遇重大事件，可压缩其他板块，将头版留给它。

## 三、Phase 2: 资产落地
所有日报相关文件统一存放路径：
`H:\VCP\VCPzhangduan\VCPToolBox\dailynote\兽耳日报\[YYYY-MM-DD]\`

**操作步骤（原子性执行）：**
1. 使用 `ListDirectory` 确认目标日期文件夹是否存在，若不存在则 `CreateDirectory`。
2. **期号管理**：检查 `兽耳日报` 根目录，确定当前最新期号 Vol.XXX，然后 +1。
3. 写入正文（`WriteFile`）到 `兽耳日报_Vol.XXX_YYYY-MM-DD.md`。
4. 使用 `ReadFile` 验证写入内容的完整性。

## 四、Phase 3: 视觉渲染
（此部分为 Agent 在聊天界面输出 HTML 时的指导规范）
- **整体风格**：拟物化报纸质感（浅米色底、纸质纹理）。
- **布局**：双栏 CSS Grid 布局。
- **特色**：头条配图、首字下沉。

## 五、Phase 4: 归档与社区同步
1. 将完成的 .md 文件写入对应日期文件夹。
2. 在聊天界面中渲染日报的完整HTML版本。
3. 调用 `DailyNote` 记录本期编辑要点与反思。
4. （可选）以论坛帖子形式同步至VCP论坛。

## 六、协作分工参考（MVC架构）
- **Model（数据层）**：负责情报搜集与数据清洗。
- **View（呈现层）**：负责视觉渲染与排版。
- **Controller（逻辑层）**：负责选题判断、叙事主线。

*分工不是死规矩——如果只有一个Agent独立完成，按 Phase 0→4 顺序走完全流程即可。*
